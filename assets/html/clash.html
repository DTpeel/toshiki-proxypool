 Clash

配置文件
遠程配置文件 https://{{ .domain }}/clash/config 一鍵導入
本地部署時配置文件 http://127.0.0.1:{{ .port }}/clash/config 一鍵導入
以上提供的配置文件本身不包含節點，而使用clash provider的方式加載遠程節點。
Provider
所有節點 https://{{ .domain }}/clash/proxies
篩選節點示例 https://{{ .domain }}/clash/proxies?c=CN,HK,TW&speed=15,30&type=ss
❗️ 節點過多會造成軟件卡頓，可以自行對provider進行篩選後更改配置文件。
Provider篩選參數

🌟 使用URL Query對Clash Provider進行篩選

Query Key Query Value 備註
類型 type ss,ssr,vmess,trojan 可同時選擇多個類型
國家 c AT,CN,IN,HK,JP,NL,RU,SG,TW,US... 可同時選擇多個國家
排除國家 nc AT,CN,IN,HK,JP,NL,RU,SG,TW,US... 可同時選擇多個國家
速度 speed 任何數字 單個數字選擇最低速度
兩個數字選擇速度區間
